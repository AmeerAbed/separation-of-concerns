Isolating JavaScript


A deep look inside program execution

Stepping Through


or "tracing"


(in class)

What is this?


To master JS you must learn how your code is executed.

To learn this, you must trace programs one step at a time.

There are two outstanding tools to help guide you ...

JS Tutor


  • + Presents just what's needed to for understanding
  • + Not just for beginners! it's helpful forever
  • - Can only study small snippets out of full projects
  • - It is limited. No DOM, events, handlers, ...
  • intro to JS Tutor

DevTools Debugger


  • + Covers everything: JS, DOM, events, network, ...
  • + Highly detailed information about your program
  • + Allows you to study your full projects in context
  • - Overwhelming until you have some experience
  • intro to Chrome Debugger
